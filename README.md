## ✦ Features

| Capability | Description |
|---|---|
| **Three-Prompt Pipeline** | Every idea generates Planning & Architecture, Implementation, and Review & Optimization prompts |
| **Smart Domain Detection** | Auto-identifies project domain (Full Stack, AI/ML, SaaS, DevOps, Mobile, etc.) |
| **Multi-Depth Support** | Standard, Detailed, or Expert-level prompt generation |
| **Target AI Selection** | Optimize for Claude, ChatGPT, Gemini, Grok, Cursor — or keep model-agnostic |
| **Expandable Cards** | Collapsible prompt cards with one-click copy |
| **Copy All** | Copy all three prompts at once, pre-formatted |
| **Local History** | Past compilations saved in your browser |
| **Mobile Responsive** | Fully functional on mobile with slide-out sidebar |

## ✦ How It Works

```
1. Describe your idea ──► 2. Configure options ──► 3. Compile ──► 4. Use the prompts
```

**1. Describe** — Type your project idea (e.g., *"Build a SaaS analytics dashboard with Next.js, Supabase, and Stripe"*)

**2. Configure** — Select output depth, domain, and target AI model

**3. Compile** — Press `Enter` or click the send button

**4. Use** — Copy individual cards or all three at once, then paste into your AI coding tool

## ✦ Prompt Output Structure

| Prompt | Focus | What You Get |
|---|---|---|
| **01 — Plan** 🎯 | Planning & Architecture | Requirements, system architecture, tech stack, database design, API design, folder structure, roadmap, security, scalability |
| **02 — Build** 🛠️ | Implementation | Complete code scaffolding, backend/frontend code, auth flows, database schemas, third-party integrations, tests |
| **03 — Optimize** 🚀 | Review & Hardening | Code quality audit, security review, performance analysis, bug detection, refactoring, CI/CD, monitoring, production scorecard |

## ✦ Getting Started

### Local Usage

```bash
git clone https://github.com/Srikar-Merugu/PromptCompiler.git
cd PromptCompiler
open index.html
```

Or serve with a local HTTP server:

```bash
python3 -m http.server 3000
open http://localhost:3000
```

### Deployment

Deploy to any static hosting provider — **Vercel**, **Netlify**, **GitHub Pages**, or **Cloudflare Pages**.
